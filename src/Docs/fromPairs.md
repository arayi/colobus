# fromPairs

## Function Description

The inverse of toPairs; this method returns an object composed from key-value pairs.

## Function Technical Explanation

Arguments

pairs (Array): The key-value pairs.
Returns

(Object): Returns the new object.

## Example

```javascript
_.fromPairs([['a', 1], ['b', 2]]);
// => { 'a': 1, 'b': 2 }
```
